 1) Daily Maximum Pairs (5)

This is only a payout limit.

Meaning:

Even if the user has 50 pairs possible today, you will pay only 5 pairs today.

The remaining pairs are not lost — they stay as unused BV.

So Daily Max Pairs controls: “How much pair commission can be paid per day.”

2) Carry Forward Mode (daily / weekly / lifetime)

This controls: “How long unused BV can remain valid.”

Because when Daily Max Pairs blocks payout, you will have leftover BV.

Carry Forward Mode decides what happens to that leftover BV:

✅ lifetime

Unused BV never expires.

Tomorrow you can still use it to make pairs.

✅ daily

At end of day, leftover BV is reset to 0 (or flushed).

Tomorrow starts fresh (no carry).

✅ weekly

Carry allowed within the same week.

When week changes, leftover BV reset.

So Carry Forward Mode is about expiry/reset, not payout speed.

3) Carry Forward Cap (BV)

This controls: “Maximum BV allowed to carry forward.”

Example:

Carry cap = 50,000 BV

If left has 90,000 BV leftover, system keeps only 50,000, and extra 40,000 is discarded (or frozen).

Why? To prevent very old users from stacking unlimited BV and breaking payouts later.
